<objective>
Implement `/discuss`, `/plan`, and `/execute` slash commands that read/write `.muonroi-flow/runs/<id>/` artifacts. `/discuss` creates runs and captures gray areas; `/plan` gates on resolved gray areas; `/execute` enters a QC-lock execution loop.

Purpose: These three commands form the core GSD workflow loop: clarify (discuss) -> plan -> execute. Each persists state to disk for cross-session continuity.
Output: Three slash command handlers registered via `registerSlash`, each with unit tests.
</objective>
