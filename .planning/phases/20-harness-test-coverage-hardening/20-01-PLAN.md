### T3 — Add `lint:harness-skips` npm script
- Create `scripts/check-harness-skips.ts` modelled on `scripts/check-semantic-wrap.ts`:
  - Glob `tests/harness/**/*.spec.ts`.
  - Regex parse `^\s*(it|describe)\.(skip|todo)\b` — count per spec, total skip+todo, total specs.
  - Ignore `describe.skipIf(...)` (legitimate platform/env guard).
  - Compute `ratio = (skip + todo) / total`.
  - Warn-only by default; exit 1 only when `--strict` flag present AND `ratio > THRESHOLD`.
  - Default threshold = 0.40 (initial calibration above current 0.39 so adding one more skip warns; tighten in a later phase as blockers clear).
- Add to `package.json` scripts: `"lint:harness-skips": "bun run scripts/check-harness-skips.ts"` and `"lint:harness-skips:strict": "bun run scripts/check-harness-skips.ts --strict"`.
