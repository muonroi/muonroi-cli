<redactor_design>
The redactor is a SINGLETON with two scrubbing layers:

LAYER 1 — Static regex set (always-on, applies to any string):
- /sk-ant-[A-Za-z0-9_-]{20,}/g           → "sk-ant-***REDACTED***"
- /sk-[A-Za-z0-9_-]{20,}/g               → "sk-***REDACTED***"  (catches OpenAI/general)
- /eyJ[A-Za-z0-9_-]+\.[A-Za-z0-9_-]+\.[A-Za-z0-9_-]+/g  → "***REDACTED-JWT***"
- /(ANTHROPIC_API_KEY|OPENAI_API_KEY|GOOGLE_API_KEY|DEEPSEEK_API_KEY)\s*=\s*\S+/g → "$1=***REDACTED***"
- /Authorization:\s*Bearer\s+\S+/gi      → "Authorization: Bearer ***REDACTED***"
- /[Xx]-[Aa]pi-[Kk]ey:\s*\S+/g           → "x-api-key: ***REDACTED***"

LAYER 2 — Enrolled live values (registered at runtime as soon as a real key is loaded):
- redactor.enrollSecret(value: string): void
  - Splits value into substrings of length >= 8 to avoid replacing common short strings
  - Stores in a Set
  - On each redact() call, replaces every enrolled value with "***REDACTED-ENROLLED***"
  - This catches Anthropic key formats that don't match the public regex (e.g. if a provider rotates format)

API surface:

```typescript
// src/utils/redactor.ts
class Redactor {
  private enrolled = new Set<string>();
  enrollSecret(value: string): void;
  redact(input: string): string;
  redactError(err: unknown): string;
  installGlobalPatches(): void;   // monkey-patches console.* to wrap their string args
  uninstallGlobalPatches(): void; // for tests
}
export const redactor: Redactor;
```

`installGlobalPatches()` wraps console.log/error/warn/info/debug. For each call, it stringifies args, runs through redact(), and forwards. Caveat: this is a process-wide side effect — it MUST be called exactly once at process start (in src/index.ts before anything else logs).

Boot order in src/index.ts:
  1. import { redactor } from "./utils/redactor";
  2. redactor.installGlobalPatches();   // FIRST line of executable code
  3. const key = await loadAnthropicKey();
  4. redactor.enrollSecret(key);        // BEFORE the next log line that might mention it
  5. ... rest of app boot
</redactor_design>

    B) `streamAnthropicMessage(req: ProviderRequest): ProviderStream` — wraps AI SDK v6 streamText with @ai-sdk/anthropic, adapts events to our StreamChunk shape:

      ```typescript
      import { streamText } from "ai";
      import { createAnthropic } from "@ai-sdk/anthropic";
      import type { ProviderRequest, ProviderStream } from "./types.js";

      export async function* streamAnthropicMessage(req: ProviderRequest): ProviderStream {
        const anthropic = createAnthropic({ apiKey: req.apiKey });
        try {
          // streamText returns a result synchronously in v6 — fullStream is the async iterator.
          const result = streamText({
            model: anthropic(req.model),
            messages: req.messages,
            abortSignal: req.abortSignal,
          });

          for await (const chunk of result.fullStream) {
            switch (chunk.type) {
              case "text-delta":
                // v6: TextStreamPart.text (string). Verified via context7 vercel/ai docs 2026-04-29.
                yield { kind: "text-delta", text: chunk.text };
                break;
              case "tool-call":
                yield { kind: "tool-call", toolCallId: chunk.toolCallId, toolName: chunk.toolName, input: chunk.input };
                break;
              case "tool-result":
                yield { kind: "tool-result", toolCallId: chunk.toolCallId, output: chunk.output };
                break;
              case "finish":
                yield { kind: "finish", reason: chunk.finishReason as any, usage: chunk.totalUsage ?? chunk.usage };
                break;
              case "error":
                yield { kind: "error", error: chunk.error instanceof Error ? chunk.error : new Error(String(chunk.error)) };
                break;
              // Other v6 events (text-start, text-end, reasoning, source, file, tool-input-*) are
              // ignored in Phase 0 — Phase 1 may surface reasoning/source for tool tier upgrades.
            }
          }
        } catch (err) {
          yield { kind: "error", error: err instanceof Error ? err : new Error(String(err)) };
        }
      }
      ```

      Field-name lock per A-3 (context7 verification 2026-04-29): `chunk.text` for text-delta, `chunk.input` for tool-call args, `chunk.output` for tool-result, `chunk.finishReason` for finish reason. NO version-conditional fallback — these are the v6 names.
